# App for my house management and for my family management   
An app for a better management of planning, shower planning, tv, shopping and some other features. I built the project by using the library Streamlit. I built this app because I wanted to see if I can build an app only using python.  
# Technologies   
- Python
# Features
Here is what you can do with App_Maison : 
- Create your own shower planning.
- Manage your entire planning.
- Attribute some task to someone and he receive a notification on his phone.
- Create some shoppings lists and add items to it.
- See what is on TV's atm.
- See what is the weather, the date and the time.
- Add a note to the family wall.
- Organize what you want to eat for this week.
